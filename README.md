# ASH
A.S.H е приложение, създадено да повиши продуктивността на учениците. То се свързва с Google Classroom, за да показва оценки, предстоящи задачи и крайни срокове на едно място. Чрез автоматично създаване на график и система за известия, A.S.H помага на учениците да планират времето си по-ефективно и да не пропускат важни задължения.

# Requirements
- Windows 10+
- 64 bit
- Google Classroom courses
